# Invoke-Speak

## SYNOPSIS

## SYNTAX

```
Invoke-Speak [-Expression] <String> [<CommonParameters>]
```

## DESCRIPTION
Use the speech synthesizer in powershell

## EXAMPLES

### EXAMPLE 1
```
Speak -Ex "Want to play a Game?"
```

### EXAMPLE 2
```
Speak "Want to play a Game?"
```

## PARAMETERS
