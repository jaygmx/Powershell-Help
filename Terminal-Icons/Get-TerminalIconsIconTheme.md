# Get-TerminalIconsIconTheme

## SYNOPSIS
List the available icon themes.

## SYNTAX

```
Get-TerminalIconsIconTheme
```

## DESCRIPTION
List the available icon themes.

## EXAMPLES

### EXAMPLE 1
```
Get-TerminalIconsIconTheme
```

Get the list of available icon themes.

## PARAMETERS

## INPUTS

### None.
## OUTPUTS

### System.Collections.Hashtable
### An array of hashtables representing available icon themes.
