# Show-TMOutputColor

## SYNOPSIS
This function will display the available colors that can be used with the Write-TMOutput function.

## SYNTAX

```
Show-TMOutputColor
```

## DESCRIPTION
This function will display the available colors that can be used with the Write-TMOutput function.
Colors can be used for either the ForegroundColor or BackgroundColor parameter of Write-TMOutput.
