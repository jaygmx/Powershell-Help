# Disable-PasswordPeek

## SYNOPSIS

## SYNTAX

```
Disable-PasswordPeek [<CommonParameters>]
```

## DESCRIPTION
Disables Password peaking in windows password prompts.
Especially useful if providing remote support and needing to type in admin credentials.
