# New-UDPolarChartOptions

## SYNOPSIS

## SYNTAX

```

 [[-TooltipOptions] <Object>] [[-xAxes] <Object>] [[-yAxes] <Object>] [[-StartAngle] <Single>]
 [[-AnimateRotate] <Boolean>] [[-AnimateScale] <Boolean>] [<CommonParameters>]
```
