# Set-ModuleVersion

 [[-ReleaseNotesPath] <String>] [[-NuspecPath] <String>] [[-Version] <Version>] [[-PreReleaseVersion] <String>]
 [[-BuildMetadata] <String>] [<CommonParameters>]
