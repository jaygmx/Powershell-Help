# New-UDTextbox

## SYNOPSIS
Creates a textbox.

## SYNTAX

```
New-UDTextbox [-Autofocus] [-Disabled] [<CommonParameters>]
```

## DESCRIPTION
Creates a textbox.

## EXAMPLES

### Example 1
```
New-UDTextbox -Label "Textbox" -Placeholder "Enter your name"
```

Creates a new textbox that has a placeholder asking to enter the user's name.
