# Disable-UDLogging

## SYNOPSIS
Disables logging for Universal Dashboard.

## SYNTAX

```
Disable-UDLogging [<CommonParameters>]
```

## DESCRIPTION
Disables logging for Universal Dashboard.

## EXAMPLES

### Example 1
```
PS C:\> Disable-UDLogging
```

Disables logging for Universal Dashboard.
