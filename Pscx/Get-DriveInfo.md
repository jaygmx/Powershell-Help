# Get-DriveInfo

## SYNOPSIS
PSCX Cmdlet: Gets disk usage information on the system's disk drives.

## SYNTAX

```
Get-DriveInfo [<CommonParameters>]
```

## DESCRIPTION
Gets disk usage information on the system's disk drives.
