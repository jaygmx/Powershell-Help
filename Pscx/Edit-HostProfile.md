# Edit-HostProfile

## SYNOPSIS
Opens the current user's profile for the current host in a text editor.

## SYNTAX

```
Edit-HostProfile
```

## DESCRIPTION
Opens the current user's profile for the current host ($Profile.CurrentUserCurrentHost) in a text editor.
