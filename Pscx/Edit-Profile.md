# Edit-Profile

## SYNOPSIS
Opens the current user's "all hosts" profile in a text editor.

## SYNTAX

```
Edit-Profile
```

## DESCRIPTION
Opens the current user's "all hosts" profile ($Profile.CurrentUserAllHosts) in a text editor.
