# Get-UserReport

## SYNOPSIS

## SYNTAX

```
Get-UserReport [<CommonParameters>]
```

## DESCRIPTION
Create a report of Active Directory users

## EXAMPLES

### EXAMPLE 1
```
Specify is where the report is to be saved.
```

Get-UserReport -path C:\UserReport.csv
