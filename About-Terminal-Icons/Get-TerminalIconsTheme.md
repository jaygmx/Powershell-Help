# Get-TerminalIconsTheme

## SYNOPSIS
Get the currently applied color and icon theme.

## SYNTAX

```
Get-TerminalIconsTheme [<CommonParameters>]
```

## DESCRIPTION
Get the currently applied color and icon theme.

## EXAMPLES

### EXAMPLE 1
```
Get-TerminalIconsTheme
```

Get the currently applied Terminal-Icons color and icon theme.
